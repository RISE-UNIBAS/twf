Context Help: Export List

The Export List page shows all exports that have been created in your project. You can use this page to monitor, download, and manage your exports.

Export List Table

The table displays the following information for each export:

  • Title: The name of the export.
  • Type: What type of content was exported (documents, collections, project, etc.).
  • Format: The file format of the export (JSON, CSV, XML, etc.).
  • Status: Current status of the export (processing, completed, failed).
  • Created: When the export was created.
  • Actions: Buttons to download or delete the export.

Managing Exports

You can:

  • Download: Click the download button to retrieve the export file.
  • Delete: Remove exports you no longer need by clicking the delete button.
  • Filter: Use the filter options to find specific exports by type or date.
