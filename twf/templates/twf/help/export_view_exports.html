Export List

The Export List page shows all exports that have been created in your project. This centralized view allows you to monitor, download, manage, and track the history of your export operations.

Export List Features

The Export List provides a comprehensive overview of all your exports:

  • Complete History: View all exports created in your project
  • Status Tracking: Monitor the progress of ongoing export operations
  • Download Access: Easily retrieve completed export files
  • Export Management: Organize, filter, and clean up your export history
  • Export Details: Access information about each export's configuration and content
  • Export Analytics: Track export patterns and usage over time

Export List Table

The main table displays detailed information for each export:

  • Title: The name or identifier of the export
  • Type: The category of content that was exported
    • Document exports (individual or multiple documents)
    • Collection exports (organized document groups)
    • Dictionary exports (terminology resources)
    • Tag exports (entity annotations and references)
    • Project exports (comprehensive project packages)
    • Zenodo publications (research data repositories)
  • Format: The file format of the export
    • Structured data formats (JSON, XML, RDF)
    • Tabular formats (CSV, Excel)
    • Document formats (PDF, DOCX, HTML)
    • Archive formats (ZIP, TAR)
    • Specialized formats (TEI, METS/MODS)
    • Custom defined formats
  • Status: Current state of the export process
    • Queued: Export is waiting to be processed
    • Processing: Export is currently being generated
    • Completed: Export has finished successfully
    • Failed: Export encountered errors
    • Canceled: Export was terminated before completion
    • Expired: Export has exceeded its retention period
  • Created: When the export was initiated
  • Completed: When the export process finished
  • Size: The file size of the completed export
  • Created By: User who initiated the export
  • Actions: Available operations for each export

Export Actions

Several actions are available for managing exports:

  • Download:
    • Retrieve the completed export file
    • Save the export to your local system
    • Access the exported content for use
  • View Details:
    • See detailed information about the export
    • View export configuration parameters
    • Check content inclusion details
    • Review error messages for failed exports
  • Repeat Export:
    • Create a new export using the same parameters
    • Regenerate exports with updated content
    • Rerun exports that failed previously
  • Delete:
    • Remove exports you no longer need
    • Clean up your export history
    • Free up storage space for new exports
  • Share:
    • Generate shareable links for specific exports
    • Set access permissions for shared exports
    • Track export access and downloads

List Management

The Export List includes several management features:

  • Filtering:
    • Filter by export type (documents, collections, etc.)
    • Filter by format (JSON, PDF, etc.)
    • Filter by status (completed, processing, failed)
    • Filter by date range or time period
    • Filter by creator or other metadata
  • Sorting:
    • Sort by any column in ascending or descending order
    • Arrange exports by creation date, size, type, etc.
    • Organize exports according to your current needs
  • Searching:
    • Search export titles or descriptions
    • Find exports containing specific content
    • Locate exports by specific parameters
  • Bulk Operations:
    • Select multiple exports for batch actions
    • Delete several exports at once
    • Download multiple exports as a package

Export Status Monitoring

The Export List provides real-time status updates:

  • Track progress of ongoing export operations
  • Receive notifications when exports complete
  • View detailed progress for large exports
  • Identify and troubleshoot failed exports
  • Monitor system performance and export queue

Best Practices

  • Regularly review and clean up your export history to manage storage
  • Use meaningful export titles to easily identify exports later
  • Download and backup important exports to your local system
  • Check failed exports for error messages to understand issues
  • Use filters to quickly locate specific exports in large projects
  • Document export parameters for significant research outputs
  • Monitor export sizes to manage storage constraints
