Document Viewer

The Document Viewer allows you to examine and work with the content of a document in your project. This view provides a comprehensive interface for viewing page content, managing document tags, and annotating documents.

Document Navigation

The document viewer provides several ways to navigate through your document:

  • Page navigation: Move forward and backward through document pages
  • Page thumbnails: Click on page thumbnails for quick navigation
  • Page selector: Jump directly to a specific page by number
  • Zoom controls: Adjust zoom level to view page content in different detail levels

Content Display

Document content is displayed with several helpful features:

  • Text recognition: Text is recognized and overlaid on page images
  • Tag highlighting: Entity tags are highlighted in different colors based on type
  • Line numbering: Text lines are numbered for easier reference
  • Searchable text: Use browser search to find specific text in the displayed content

Working with Tags

Within the document viewer, you can:

  • View tags: See all tags identified in the current page
  • Select tags: Click on highlighted tags to select them
  • Review tag details: See tag information and status
  • Resolve tags: Assign tags to dictionary entries directly from the viewer
  • Create new tags: Select text to create new manual tags
  • Edit existing tags: Modify tag attributes or boundaries

Document Information

The document viewer provides access to various document metadata:

  • Document title: The name and identifier of the current document
  • Processing status: Current state of the document's workflow
  • Document metadata: Access to key document information
  • Tag statistics: Summary of tag counts by type and status
  • Processing history: Record of operations performed on the document

Sidebar Tools

The sidebar provides access to additional functionality:

  • Tag List: View and filter all tags in the current document
  • Page List: Access all pages with thumbnail previews
  • Metadata: View and edit document and page metadata
  • Notes: Add and view document annotations and comments
  • Dictionary Lookup: Search dictionaries for terms in the document

Annotation Tools

The document viewer offers several ways to annotate and mark up documents:

  • Highlighting: Mark important text sections with color highlighting
  • Comments: Add notes to specific document sections
  • Bookmarks: Mark important pages for quick access
  • Tag creation: Select text to create new entity tags

Search and Filter

The document viewer includes several search capabilities:

  • Text search: Find specific text throughout the document
  • Tag filter: Filter displayed tags by type, status, or other attributes
  • Jump to matches: Navigate directly to search matches

Best Practices

  • Use keyboard shortcuts for faster navigation (arrow keys, Page Up/Down)
  • Adjust zoom levels appropriately when working with dense text or images
  • Review tags in context rather than just in the tag list
  • Add notes to document your analysis and decision-making process
  • Use the tag filter to focus on specific tag types when working through a document
  • Save your work regularly, especially after making extensive tag changes
