Context Help: Create Document

The Create Document page allows you to manually add new documents to your project. While most documents are typically imported from Transkribus exports, this feature enables you to create additional documents when needed.

Document Creation Process

  1. Basic Information: Enter essential document details:
    • Document title
    • Document ID (unique identifier)
    • Description
  2. Document Content: Add document content:
    • Upload document files (if applicable)
    • Enter or paste text content
    • Configure page structure
  3. Metadata: Add descriptive metadata:
    • Author information
    • Creation date
    • Categories and keywords
    • Custom metadata fields
  4. Document Settings: Configure document properties:
    • Workflow status
    • Access permissions
    • Initial tag settings
  5. Create Document: Save the new document to your project

Use Cases for Manual Document Creation

This feature is particularly useful for:

  • Supplementary Materials: Adding contextual documents not in Transkribus
  • Research Notes: Creating documents to capture analysis and observations
  • Composite Documents: Assembling content from multiple sources
  • Test Documents: Creating sample content for workflow testing
  • External Content: Incorporating material from sources outside Transkribus

Content Options

When creating a document, you can include:

  • Plain Text: Basic text content without formatting
  • Structured Text: Text with paragraph and section markers
  • Uploaded Files: Digital documents in supported formats
  • Page References: Links to external content

Best Practices

  • Use clear, descriptive titles that indicate document content
  • Assign unique document IDs following project conventions
  • Include comprehensive metadata for better searchability
  • Structure content into logical pages when appropriate
  • Add sufficient description to explain the document's purpose and context
  • Consider how manually created documents will integrate with imported content

After Creation

Once you've created a document, you can:

  • Add or edit pages within the document
  • Extract and normalize tags
  • Include the document in collections
  • Apply the same workflows used for imported documents
