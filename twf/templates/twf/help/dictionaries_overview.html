Context Help: Dictionaries Overview

The Dictionaries Overview page provides a central hub for managing all dictionaries in your project. Dictionaries in TWF are collections of terms and their standardized forms, which help normalize and enrich tags in your documents.

What Are Dictionaries?

In TWF, dictionaries serve as controlled vocabularies that:

  • Help standardize variations of the same term (e.g., different spellings of names)
  • Provide structured information about entities (people, places, organizations, etc.)
  • Link to external authority records (like GND, Wikidata, GeoNames)
  • Improve search and analysis by using normalized terms

Dictionary Types

TWF supports several dictionary types:

  • Person Dictionaries: For names of individuals
  • Place Dictionaries: For geographic locations
  • Organization Dictionaries: For institutions and groups
  • Date Dictionaries: For standardizing date formats
  • General Dictionaries: For other types of terms
  • Custom Dictionaries: For project-specific needs

Overview Page Features

From this page, you can:

  • View All Dictionaries: See a list of all dictionaries in your project
  • Dictionary Statistics: View the number of entries in each dictionary
  • Quick Navigation: Access different dictionary management functions
  • Search/Filter: Find specific dictionaries by type or name

Key Actions

Main actions available from this page:

  • Browse Dictionaries: View and manage existing dictionaries
  • Add Dictionary: Add existing dictionaries to your project
  • Create Dictionary: Create a new dictionary from scratch
  • Access workflows for batch enrichment of dictionaries using external sources

Dictionary Workflows

TWF offers several ways to work with dictionaries:

  • Manual Management: Create and edit entries manually
  • Automated Batch Processing: Enrich multiple entries at once using AI or external databases
  • Supervised Enrichment: Review and approve suggested enrichments one by one
  • Import/Export: Share dictionaries between projects or with other systems
