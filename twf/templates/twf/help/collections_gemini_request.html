Collection Processing with Gemini (Supervised)

This feature allows you to process individual collection items with Google's Gemini AI while maintaining direct supervision over the analysis. Supervised processing gives you more control over how Gemini interprets and analyzes your collection items.

What is Supervised Collection Processing?

Unlike batch processing which runs automatically on multiple items, supervised processing:

  • Processes one collection item at a time
  • Gives you the ability to review and modify Gemini's suggestions
  • Allows interactive refinement of the AI analysis
  • Provides immediate feedback on the results
  • Lets you adjust prompts based on initial responses
  • Gives you complete control over which suggestions to accept

Prerequisites

Before using this feature, you need:

  • A valid Google AI Studio API key configured in your project settings
  • A collection with properly configured items
  • Defined prompts that specify what Gemini should analyze or extract

Supervised Processing Steps

  1. Select the collection item you want to process
  2. Choose or create a prompt template
  3. Configure processing parameters (model, temperature, etc.)
  4. Submit your request to Gemini
  5. Review the AI's response
  6. Accept, modify, or reject the suggestions
  7. Save the final results to your collection item

When to Use Supervised Processing

Supervised processing with Gemini is ideal when:

  • Working with multilingual or diverse format documents
  • Processing collection items with both textual and visual elements
  • Handling documents that require careful interpretation
  • Developing and refining prompts before batch processing
  • Working with specialized or technical content
  • Training team members on effective AI interaction

Configuration Options

When setting up a supervised Gemini process, you can configure:

  • Model: Select which Gemini model to use
  • Temperature: Control creativity vs. determinism (0.0-1.0)
  • Top P/Top K: Parameters that influence output variety
  • Max Output Tokens: Limit the length of AI responses
  • Processing Scope: Whether to process the entire document or specific sections
  • Result Format: How Gemini should structure its response

Refining Your Prompts

Supervised processing is an excellent opportunity to refine your prompts:

  • Experiment with different instruction phrasings
  • Adjust the level of detail requested
  • Try different output formats
  • Add or remove examples
  • Incorporate domain-specific terminology
  • Take advantage of Gemini's multilingual capabilities
  • Save successful prompts for future use in batch processing

Best Practices

  • Review the collection item thoroughly before processing
  • Start with clear, specific instructions in your prompt
  • Be explicit about the format you want for Gemini's response
  • Use the interactive interface to refine responses if needed
  • Save particularly effective prompts to your project's prompt library
  • Document any modifications you make to Gemini's suggestions
  • Consider supervised processing as a training phase before batch processing
