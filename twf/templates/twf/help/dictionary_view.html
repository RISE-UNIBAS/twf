Dictionary View

This page displays a specific dictionary and its entries. You can browse, search, filter, and manage the dictionary's contents.

Dictionary Overview

  • Dictionary title, type, and description
  • Number of entries and statistics
  • Creation and modification information

Entry Management

  • Search for specific entries using the search box
  • Filter entries by various criteria
  • Sort entries by clicking on column headers
  • Add new entries using the "Add Entry" button
  • Edit or delete existing entries using the action buttons

Batch Operations

You can perform operations on multiple entries at once:

  • Export selected entries or the entire dictionary
  • Normalize multiple entries using external authorities
  • Apply bulk edits to selected entries
