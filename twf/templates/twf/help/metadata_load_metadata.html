Context Help

This page allows you to load metadata from a file and match it to the documents or pages in your project.

Metadata File:

The file must be a json file and adhere one of the following formats:

List format: A list of dictionaries, where each dictionary represents a document or page and contains metadata keys and values. Each object in the list must have a unique identifier pointing to the document or page. 

        
[
    {
      "some_doc_id": "123456",
      "title": "Example Document",
      "author": "John Doe",
    },
    {
      "some_doc_id": "123457",
      "title": "Another Document",
      "author": "Jane Doe",
    }
]

Dictionary format: A dictionary of dictionaries, where each dictionary represents a document or page and contains metadata keys and values. The keys of the outer dictionary must be unique identifiers pointing to the document or page. 

        
{
    "123456": {
        "title": "Example Document",
        "author": "John Doe",
    },
    "123457": {
        "title": "Another Document",
        "author": "Jane Doe",
    }
}
