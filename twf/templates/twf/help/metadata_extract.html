Extract Metadata

This page allows you to automatically extract metadata from your document content using AI-powered analysis. The extraction process can identify key information from document text and create structured metadata entries.

What Can Be Extracted?

Depending on your document content, the AI can typically extract:

  • Document titles and subtitles
  • Author information and contributor names
  • Creation and publication dates
  • Location information
  • Document type and classification
  • Subject matter and keywords
  • Organizational or institutional affiliations
  • Other contextual information present in the text

Extraction Process

The metadata extraction works in the following way:

  1. Select the documents from which you want to extract metadata
  2. Choose which metadata fields you want to extract
  3. Configure the AI extraction settings and parameters
  4. Initiate the extraction process
  5. Review and refine the extracted metadata
  6. Confirm and save the results to your documents

AI Service Options

The system supports multiple AI services for metadata extraction:

  • OpenAI: Uses advanced language models to identify and extract metadata
  • Claude: Anthropic's AI assistant with strong natural language understanding
  • Gemini: Google's AI system with capabilities for analyzing document content

Each service has different strengths and may perform better with certain document types. You'll need to have the appropriate API credentials configured in your project settings.

Extraction Settings

You can customize the extraction process by:

  • Setting confidence thresholds for extracted information
  • Defining extraction patterns or templates for specific document types
  • Configuring how to handle uncertain or conflicting data
  • Specifying preferred formats for extracted dates, names, or other fields
  • Setting limits on the amount of text analyzed per document

Best Practices

  • Start with a small batch of documents to test extraction accuracy
  • Review AI-extracted metadata carefully before bulk importing
  • Use extraction on documents with clear, well-structured content for best results
  • Combine AI extraction with manual review for optimal quality
  • Consider document language when selecting AI services (some perform better with specific languages)
  • Save extraction templates for future use with similar document collections
