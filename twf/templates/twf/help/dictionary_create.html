Create New Dictionary

This page allows you to create a new dictionary from scratch. Dictionaries help normalize and enrich tags in your documents.

Required Information

  • Dictionary Title: A clear, descriptive name
  • Dictionary Type: Person, Place, Organization, Date, or General
  • Description: Brief explanation of the dictionary's purpose
  • Access Settings: Visibility and edit permissions

Dictionary Types

  • Person: For standardizing people's names (links to GND)
  • Place: For geographic locations (links to GeoNames)
  • Organization: For institutions and formal groups
  • Date: For standardizing date formats and references
  • General: For any other terms

After Creation

Once created, you can add entries manually or through batch processes, enrich them using external sources, and use the dictionary for tag normalization.
