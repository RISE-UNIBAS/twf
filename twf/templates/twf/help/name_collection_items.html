Name Collection Items

This feature allows you to automatically generate or update titles for collection items based on document metadata or content. Properly named collection items improve organization, searchability, and usability of your collections.

Naming Collection Items

Collection item naming is useful when:

  • You've imported documents in bulk and need meaningful titles
  • You want to standardize naming conventions across your collection
  • You need to generate titles based on document metadata
  • You want to reflect document content in item titles
  • You're preparing collections for sharing or publication

Naming Options

The system offers several approaches to naming collection items:

  • Metadata-based naming: Generate titles from document metadata fields
  • Content-based naming: Create titles based on document content analysis
  • Pattern-based naming: Apply a consistent pattern with variables
  • Sequential naming: Add sequential numbers to a base name
  • Date-based naming: Incorporate document dates into titles

Naming Process

To name your collection items:

  1. Select the collection containing the items you want to name
  2. Choose a naming method and configure its parameters
  3. Preview the generated names
  4. Make any manual adjustments needed
  5. Apply the naming changes to your collection

Metadata-Based Naming

When using metadata for naming, you can:

  • Select which metadata fields to include in the title
  • Define the order and formatting of field values
  • Specify separators between field values
  • Set fallback values for missing metadata
  • Apply transformations to field values (uppercase, lowercase, etc.)

Pattern Examples

Some common naming patterns include:

  • {author} - {title} ({year})
  • Document {sequence_number}: {title}
  • {collection_name}: {document_type} - {date}
  • {category}/{subcategory}/{document_id}

Best Practices

  • Use consistent naming patterns across related collections
  • Include the most important identifying information in titles
  • Keep titles concise but descriptive
  • Consider how titles will be used in exports and displays
  • Avoid special characters that might cause issues in file systems
  • Include date information in a standardized format
  • Back up your collection before applying bulk naming changes
