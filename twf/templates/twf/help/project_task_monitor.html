Task Monitor

The Task Monitor provides a centralized dashboard to track and manage all tasks running in your project. Tasks are operations which might take a longer time to complete, such as AI processing, or manual tasks that are performed by users. The Task Monitor allows you to monitor the status of these tasks.

Task Monitor Features

  • Real-time Status: View the current status of each task (pending, running, completed, failed).
  • Progress Tracking: See detailed progress bars for long-running tasks.
  • Task Details: View information about each task, including:
    • Task type and description
    • Start and estimated completion time
    • User who initiated the task
    • Resources being used
  • Task Management: Cancel running tasks or remove completed ones.

Task Types

The Task Monitor tracks various background operations, including:

  • Data import and export processes
  • AI batch processing operations
  • Dictionary enrichment tasks
  • Tag extraction and normalization

Task Actions

From the Task Monitor, you can:

  • Filter Tasks: Use the search and filter options to find specific tasks or task types. This only affects the display and doesn't change any tasks.
  • Cancel Tasks: Stop a running task that is no longer needed or is consuming too many resources.
  • Remove Tasks: Clear completed or failed tasks from the list. This only removes the task entry from the monitor and does not undo any work the task has already performed.
  • View Details: Get more information about a specific task, including any error messages for failed tasks.

Task Statuses

  • Pending: Task is queued but not yet running.
  • Running: Task is currently being executed.
  • Completed: Task has finished successfully.
  • Failed: Task encountered an error and could not complete.
  • Cancelled: Task was manually stopped before completion.
