Authority Batch Enrichment

This feature allows you to automatically enrich multiple dictionary entries with data from external authority services, improving data quality and adding structured metadata.

Batch Enrichment Process

  1. Choose the dictionary to enrich (matching the authority service's focus)
  2. Select entries to process (all entries or a filtered subset)
  3. Configure enrichment parameters (search precision, thresholds, etc.)
  4. Start the batch process and monitor progress
  5. Review results after completion

Enrichment Results

After processing, entries will be categorized as:

  • Successfully Enriched: Entries confidently matched and updated
  • Uncertain Matches: Entries with potential matches requiring review
  • No Matches: Entries for which no suitable records were found

Best Practices

  • Use specific, clear entry names to improve match quality
  • Start with small batches to test the matching quality
  • Review uncertain matches manually after processing
  • Use supervised workflows for ambiguous or complex entries
  • Consider the authority's regional or topical focus for best results
