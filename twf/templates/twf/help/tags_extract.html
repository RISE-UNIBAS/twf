Context Help: Extract Tags

The Extract Tags page allows you to identify and extract named entities and other significant terms from your documents. This process creates tags that can be normalized, grouped, and analyzed throughout your project.

Tag Extraction Process

  1. Document Selection: Choose which documents to process for tag extraction:
    • All documents in the project
    • Documents matching specific criteria
    • Individually selected documents
  2. Extraction Configuration: Set parameters for the extraction process:
    • Tag types to extract (people, places, organizations, dates, etc.)
    • Extraction precision settings
    • Minimum confidence threshold
    • Processing options (overwrite existing tags, append new ones)
  3. Start Extraction: Begin the automated tag identification process
  4. Review Results: After completion, view a summary of extracted tags

Extraction Methods

TWF supports several methods for tag extraction:

  • Pattern-Based: Identify tags based on text patterns and regular expressions
  • Dictionary-Based: Match text against existing dictionary entries
  • AI-Assisted: Use language models to identify named entities
  • Hybrid Approaches: Combine multiple methods for better coverage

Tag Types

The extraction process can identify various types of tags:

  • Person: Names of individuals
  • Place: Geographic locations
  • Organization: Institutions, companies, and formal groups
  • Date: Temporal references and calendar dates
  • Event: Named historical events
  • Custom Types: Project-specific categories you define

Best Practices

  • Run tag extraction early in your project workflow
  • Start with conservative settings and refine as needed
  • Create appropriate dictionaries before extraction for better results
  • Consider running extraction in phases by tag type
  • Review extracted tags to ensure quality and coverage
  • Re-run extraction if document content changes significantly

After Extraction

Once tags are extracted, you can:

  • View all extracted tags in the tag views
  • Normalize tags using dictionaries
  • Group related tags together
  • Analyze tag distributions and patterns
  • Generate tag-based statistics and visualizations
