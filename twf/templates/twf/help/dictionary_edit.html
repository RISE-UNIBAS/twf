Editing a Dictionary

This page allows you to modify the properties of an existing dictionary.

Editable Properties

  • Dictionary Title: You can rename the dictionary
  • Description: Update the dictionary's purpose and contents
  • Access Settings: Change who can view and edit this dictionary

Important Notes

  • The dictionary type (Person, Place, etc.) cannot be changed after creation
  • Renaming a dictionary does not affect its entries or their associations
  • Changes to access settings take effect immediately
