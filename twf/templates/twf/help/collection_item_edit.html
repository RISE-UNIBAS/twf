Context Help

This page allows you to update and refine a collection item. A collection item consists of structured text segments from a document, which you can review, split, and manage.

1. Title

The title field contains a system-generated name for the collection item, which you can edit if needed. Maximum length is 255 characters.

2. Document Configuration

  • This section displays the different parts of the document or page that belong to this collection item.
  • Each segment can be removed individually. This will not affect the document or page.
  • Splitting Items: You can split a collection item by clicking "Split here and create new item" between segments. This creates a separate collection item from the split section onward.

3. Collection Item Status

  • The current status of the collection item is displayed at the top.
  • You can update the status using the following buttons:
    • Set Status Open: Default state for an item that has not yet been reviewed.
    • Set Status Reviewed: Marks the item as checked and verified.
    • Set Status Faulty: Indicates that the collection item contains errors or needs revision.

4. Collection Item Options

  • View Document: Opens the full document from which the collection item is extracted.
  • Copy This Item: Duplicates the current collection item.
  • Delete This Item: Permanently deletes the collection item. âš  This action cannot be undone.

5. Notes on Item

This section allows reviewers to leave comments or notes about the collection item.
