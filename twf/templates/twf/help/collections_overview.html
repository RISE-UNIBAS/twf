Context Help: Collections Overview

The Collections Overview page provides a central access point for all collection-related functions in your project. Collections allow you to organize documents, pages, or annotations into meaningful groups for analysis and export.

What Are Collections?

In TWF, collections serve as:

  • Thematic groupings of related content
  • Custom selections of documents, pages, or text segments
  • Organizational units for analysis and export
  • Ways to structure your research material
  • Containers for creating coherent publications

Collection Management Features

From this page, you can access:

  • Your Collections: Browse existing collections in your project
  • Create Collection: Start a new collection from scratch
  • Review Collections: Access the review workflow for collections
  • Automated Workflows: Use AI tools to process collections
  • Manual Workflows: Manually manage collection content

Collection Statistics

The overview page displays key information about collections in your project:

  • Total number of collections
  • Collections by type and status
  • Recently modified collections
  • Item counts across collections

Collection Workflow Options

TWF offers several ways to work with collections:

  • Manual Creation: Carefully select and organize items yourself
  • AI-Assisted Creation: Use AI to suggest content groupings
  • Supervised Workflows: Review and approve AI-suggested content
  • Batch Operations: Process multiple collection items at once

Collection Use Cases

Collections are useful for:

  • Organizing documents by theme, time period, or subject
  • Creating curated sets of content for analysis
  • Preparing coherent groups of content for export
  • Collaborating on specific subsets of your project
  • Creating publication-ready content selections

Best Practices

  • Create collections with clear, focused purposes
  • Use descriptive names and add detailed descriptions
  • Review collection items before finalizing
  • Consider relationships between collections
  • Keep collections to a manageable size
