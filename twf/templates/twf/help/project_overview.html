Project Overview

The Project Overview provides a comprehensive summary of your current project, including key information, statistics, and recent activity.

Project Information Header

  • Project Name: Displayed at the top of the page (e.g., "Project 'Letter Test'").
  • Description: A brief description of your project's purpose.
  • Transkribus Collection ID: If your project is linked to a Transkribus collection, the ID is shown here.
  • Creation & Membership Details: Shows who created the project, when it was created, current members, and when it was last modified.

Key Metrics Summary

The overview cards provide at-a-glance information about your project:

  • Documents: Total number of documents in this project.
  • Pages: Total number of pages across all documents.
  • Users: Number of users with access to this project.

Project Statistics

This section is divided into multiple areas with detailed statistics:

Documents
  • Total Documents: Number of documents currently in the project.
  • Total Pages: Number of pages across all documents.
  • Avg. Pages/Document: The average number of pages per document.
  • Ignored Pages: Number and percentage of pages marked as ignored.
Tags
  • Total Tags: Total number of tags extracted from all documents.
  • Tags per Page: Average number of tags per page.
  • Dictionaries Connected: Number of dictionaries linked to this project.
Collections
  • Total Collections: Number of collections created in this project.
  • Collection Items: Total number of items across all collections.
  • Item Progress: Visual progress bar showing the status of collection items.
Task Activity
  • Total Tasks: Total number of tasks initiated in this project.
  • Completed Tasks: Number of successfully completed tasks.
  • Pending Tasks: Number of tasks currently in progress.
  • Failed Tasks: Number of tasks that encountered errors.

Project Setup & Recent Tasks

  • Project Setup: Shows the setup status of your project with a checklist of completed steps.
  • Recent Tasks: Lists the most recent processing tasks with their status and duration.
  • View All Tasks: Click this button to see a complete history of all project tasks.

Navigation

Use the left sidebar to navigate to other sections of the project. Each section provides specialized tools for working with different aspects of your data.

Tip: The Project Overview is an excellent starting point to monitor your project's progress and identify areas that need attention.
