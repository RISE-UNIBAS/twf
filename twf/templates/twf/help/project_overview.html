Project Overview

The Project Overview provides a summary of the selected project, including metadata, statistics, and key metrics.

Project Details

  • Project Name: Displays the name of the project.
  • Description: Shows a short project description, if available.
  • Transkribus Collection ID: If linked to Transkribus, this identifier is displayed.
  • Creator & Last Modified: Shows who created the project and when it was last updated.

Statistics

Below the project details, you can find key statistics:

  • Documents: Total number of documents in this project.
  • Pages: Total number of pages across all documents.
  • Users: Number of users with access to this project.

Data Counts

This section provides additional insights into the project's data:

  • Document Count: Number of documents currently in the project.
  • Page Count: Number of pages across all documents.
  • Average Pages per Document: The system calculates an average value for reference.

Actions

From this page, you can navigate to other sections of the project using the left sidebar. If you need to manage settings, exports, or tasks, use the links on the left.

Need more help? Click the sections in the left menu for additional options.
