Your Collections

This page displays all collections associated with your project. Collections allow you to organize your documents into meaningful groups for more effective management, analysis, and processing.

Understanding Collections

Collections in Transkribus Workflow provide a way to:

  • Organize documents into logical groupings
  • Manage related documents as a cohesive unit
  • Apply batch operations to multiple documents
  • Track progress on document processing workflows
  • Share and collaborate on specific document sets
  • Create meaningful exports of related materials

Collection List Features

On this page, you can:

  • View all collections in your project
  • Sort and filter collections by various criteria
  • Search for specific collections by name or content
  • See key metrics about each collection (item count, status, etc.)
  • Access collection management functions
  • Create new collections to organize your documents

Collection Information

For each collection, you can see:

  • Title: The name of the collection
  • Description: A brief overview of the collection's contents or purpose
  • Item Count: The number of documents in the collection
  • Created: When the collection was created and by whom
  • Status: The overall progress state of the collection
  • Actions: Available operations for this collection

Collection Actions

From this page, you can perform several actions on your collections:

  • View: Open a collection to see its contents
  • Edit: Modify collection properties and settings
  • Add Items: Include additional documents in the collection
  • Process: Apply batch operations to collection items
  • Export: Create exports from collection contents
  • Delete: Remove a collection (without deleting the documents themselves)

Creating New Collections

To create a new collection, click the "Create Collection" button. You'll need to provide:

  • A descriptive title for your collection
  • An optional detailed description
  • Initial documents to include (optional)
  • Any collection-specific settings or configurations

Collection Organization Strategies

Consider these approaches to organizing your collections:

  • Thematic grouping: Documents related to a specific subject or theme
  • Chronological grouping: Documents from a particular time period
  • Source-based grouping: Documents from the same archive or repository
  • Project phase grouping: Documents requiring similar processing stages
  • Workflow-based grouping: Documents that will undergo the same operations
