Parked Tags

This page shows all tags that have been marked as "parked" - meaning they have been set aside temporarily without being resolved. Parked tags are those that you've decided to handle later for various reasons.

What are Parked Tags?

Parked tags are a special status for tags that:

  • Require further research or consideration
  • Cannot be immediately resolved due to missing information
  • Are waiting for a decision from another team member
  • Are temporarily set aside to focus on higher-priority tags

Working with Parked Tags

In this view, you can:

  • View details of any parked tag
  • Filter tags by various criteria including type, document, and more
  • Un-park tags to return them to the open tags list
  • Resolve tags if you're now ready to assign them to a dictionary
  • View tag context to see where the tag appears in the document

Tag Information

For each parked tag, you can see:

  • Tag Text: The actual text content of the tag
  • Type: The category of the tag (person, place, etc.)
  • Document: The document containing this tag
  • Page: The page number where the tag appears
  • Actions: Options to manage this tag

Managing Parked Tags

Best practices for parked tags include:

  • Regularly review parked tags to ensure they don't get forgotten
  • Add notes to parked tags to explain why they're parked and what information is needed
  • Use the filter feature to focus on specific types of parked tags
  • Consider setting up a scheduled review of parked tags
