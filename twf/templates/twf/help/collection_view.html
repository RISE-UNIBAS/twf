Viewing a Collection

This page displays the contents and details of a collection. Collections organize related documents or document segments for processing, analysis, and export.

Collection Information

At the top of the page, you'll find the collection title, description, and basic metadata like creation date and owner.

Collection Items

The main section displays a table of all items within this collection. For each item, you can see:

  • Title: The identifier for each collection item
  • Status: The current state (Open, Reviewed, Faulty)
  • Content Preview: A brief excerpt of the item's content
  • Actions: View, edit, delete and other operations

Management Options

  • Edit Collection: Modify the collection's properties
  • Add Items: Add existing documents to the collection
  • Filter/Search: Find specific items in the collection
