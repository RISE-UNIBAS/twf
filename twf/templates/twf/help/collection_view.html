Viewing a Collection

This page displays the contents and details of a specific collection. Collections organize related documents or document segments into meaningful groups for processing, analysis, and export.

Collection Information

At the top of the page, you'll find essential information about the collection:

  • Title: The name of the collection
  • Description: A detailed explanation of the collection's purpose and contents
  • Type: The category or classification of the collection
  • Created By: The user who created the collection
  • Creation Date: When the collection was first created
  • Last Modified: When the collection was most recently updated
  • Statistics: Overview of item counts and status distribution

Collection Items Table

The main section of the page displays a table of all items within this collection:

  • Title/ID: The identifier for each collection item
  • Status: The current state of the item (Open, Reviewed, Faulty)
  • Document: The source document for this item
  • Content Preview: A brief excerpt of the item's content
  • Annotations: Number and types of annotations applied to the item
  • Created/Modified: Timestamps for item creation and updates
  • Actions: Available operations for each item

Table Controls

Above the table, you'll find controls to manage your view:

  • Search: Filter items by entering text to match item content or metadata
  • Filters: Narrow the displayed items by status, document, or other attributes
  • Sorting: Change the order of items by clicking column headers
  • Pagination: Navigate through multiple pages of items
  • Display Options: Adjust how many items are shown per page

Item Actions

For each collection item, you can perform several actions:

  • View Item: Open a detailed view of the collection item
  • Edit Item: Modify the collection item's content or settings
  • Delete Item: Remove the item from the collection
  • View Source Document: Open the original document this item was created from
  • Copy Item: Create a duplicate of this item
  • Change Status: Update the workflow status of the item

Collection Management

At the top of the page, you'll find buttons for overall collection management:

  • Edit Collection: Modify the collection's properties and settings
  • Add Items: Create new items or add existing documents to the collection
  • Process Collection: Apply AI or batch operations to multiple items
  • Export Collection: Generate an export of the collection and its contents
  • Delete Collection: Remove the entire collection and its items
  • Download Report: Generate a summary report of the collection

Status Indicators

Collection items are marked with status indicators:

  • Open: Items that have not yet been fully processed or reviewed
  • Reviewed: Items that have been verified and approved
  • Faulty: Items that have issues requiring attention
  • Reserved: Items currently being edited by another user

Best Practices

  • Use the search and filter options to quickly locate specific items
  • Review the collection's overall status distribution to track progress
  • Work through items methodically, marking them as Reviewed when complete
  • Use the "Add Items" function to grow collections incrementally
  • Export collections when you need to share or archive the organized content
  • Update collection descriptions as its purpose or scope evolves
