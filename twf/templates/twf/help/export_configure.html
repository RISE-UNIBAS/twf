Context Help: Export Configuration

The Export Configuration page allows you to set up and manage global export settings for your project. These settings will be applied to all exports created within the project.

Configuration Options

On this page, you can configure:

  • Default Export Formats: Define the preferred formats for different types of content (documents, collections, etc.).
  • Metadata Fields: Specify which metadata fields should be included in exports.
  • File Structure: Configure how exported files should be organized and structured.
  • Output Templates: Define templates for formatting the exported content.

Global vs. Individual Export Settings

Settings configured here apply project-wide but can be overridden for individual exports when needed. This allows for consistent exports across the project while maintaining flexibility for special cases.

Tips for Export Configuration

  • Consider the needs of your target audience when selecting export formats.
  • Include only essential metadata to keep exports clean and focused.
  • Save common export configurations as templates if you perform similar exports regularly.
  • Test your export configuration with a small sample before exporting large datasets.
