Context Help: Export to Zenodo

The Export to Zenodo page allows you to publish your project data to Zenodo, an open-access repository operated by CERN. This feature helps make your research data openly available, citable, and compliant with FAIR principles.

What is Zenodo?

Zenodo is a general-purpose open repository that allows researchers to deposit research papers, data sets, research software, reports, and any other research-related digital artifacts. It assigns a DOI (Digital Object Identifier) to all publicly available uploads, making your data citable.

Export Process

  1. Configure Metadata:
    • Title and description of your dataset
    • Authors and contributors
    • Keywords, subjects, and related information
    • Choose an appropriate license
  2. Select Content: Choose what to include in your Zenodo publication:
    • Documents
    • Collections
    • Dictionaries
    • Project metadata
  3. Export Format: Select how your data should be formatted for Zenodo.
  4. Publication Type: Choose whether to create a new publication or update an existing one.
  5. Review and Submit: Verify your export settings before publishing to Zenodo.

Zenodo API Credentials

To use this feature, you need to have Zenodo API credentials configured in your project settings. If you don't see these options, please check your project's credential settings or contact your project administrator.

Publication Tips

  • Provide thorough metadata to make your dataset more discoverable.
  • Choose an appropriate license that reflects how you want others to use your data.
  • Consider creating a README file to include with your export that explains the dataset structure.
  • You can publish to Zenodo Sandbox for testing before doing a final publication.
