Project Management

The Project Management view allows you to manage your projects by closing or deleting them.

Managing Projects

Each project listed on this page includes:

  • Project Name: The title of the project.
  • Description: A brief description of the project (if provided).
  • Created: The date and time the project was created and by whom.
  • Last Updated: The last time any modifications were made to the project.
  • Status: Indicates whether the project is open or closed.

Actions

For each project, you have the following actions available:

  • Delete Project: Permanently removes the project. This action cannot be undone.
  • Close Project: Marks the project as closed, preventing further modifications.

Additional Options

You can also navigate to other administrative functions using the sidebar:

  • Select Project: View and switch between different projects.
  • User Options: Manage user accounts, profiles, and passwords.
  • Administrator Options: Access advanced settings, including project creation and administrative tools.

Need more help? If you are unsure about closing or deleting a project, contact an administrator.
