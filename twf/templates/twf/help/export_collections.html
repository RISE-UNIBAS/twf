Context Help: Export Collections

The Export Collections page allows you to export collections from your project in various formats. Collections organize related documents, pages, or annotations that you've grouped together.

Export Process

  1. Select Collections: Use the table to select which collections to include in the export.
  2. Choose Format: Select the desired export format (JSON, XML, CSV, TXT, etc.).
  3. Configure Options: Set specific options for the selected format, such as:
    • Including collection items' content
    • Including metadata and annotations
    • Formatting options
    • File organization preferences
  4. Start Export: Click the export button to begin the export process.

Collection Content Options

When exporting collections, you can choose to include:

  • Collection Metadata: Information about the collection itself, such as title, description, and creation date.
  • Item Metadata: Metadata associated with each item in the collection.
  • Item Content: The actual content of documents, pages, or annotations in the collection.
  • Annotations: Any annotations or notes added to collection items.
  • Relationships: Connections between items in the collection.

Export Format Considerations

Different formats offer different advantages:

  • JSON: Preserves collection structure, relationships, and all metadata.
  • XML: Good for interoperability with other systems while maintaining structure.
  • CSV/TXT: Simplified formats, useful for basic data analysis or text review.
