Viewing a Collection Item

This page displays the details of a collection item. A collection item consists of structured text segments from a document that have been grouped together.

Collection Item Details

  • Title: The name of the collection item
  • Status: The current workflow state of the item
  • Document: The source document from which this item was created

Content Segments

The main section displays the text segments that make up this collection item. Each segment is shown with its context and formatting preserved from the original document.

Available Actions

  • Edit Item: Modify the collection item's properties or content
  • View Document: See the full source document
  • Download: Export the collection item as text or JSON
