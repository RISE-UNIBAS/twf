Collection Processing (Supervised)

This feature allows you to process individual collection items with AI while maintaining direct supervision. Unlike batch processing, supervised processing gives you control over how each item is analyzed.

How It Works

Unlike batch processing which runs automatically, supervised processing:

  • Processes one collection item at a time
  • Allows you to review and modify AI suggestions
  • Provides immediate feedback on results
  • Gives you control over which suggestions to accept

Processing Steps

  1. Select the collection item to process
  2. Choose or create a prompt template
  3. Submit your request to the AI
  4. Review the AI's response
  5. Accept, modify, or reject the suggestions
  6. Save the final results
