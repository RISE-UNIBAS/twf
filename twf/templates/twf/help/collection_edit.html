Editing a Collection

This page allows you to modify the properties and settings of an existing collection. A collection in TWF serves as a container for organizing related document items.

Collection Properties

You can edit the following collection properties:

  • Title: The display name of the collection, which should be descriptive and unique
  • Description: A detailed explanation of the collection's purpose and contents
  • Collection Type: The category or classification of this collection
  • Status: The current state of the collection (e.g., in progress, complete)

Collection Management

From the edit page, you can also:

  • Change ownership: Transfer the collection to another project member
  • Set visibility: Control which project members can access this collection
  • Define workflows: Set up processing steps for items in this collection
  • Configure sorting: Set the default sort order for collection items

After Saving Changes

When you save your changes, you will be returned to the collection view page where you can see all items within this collection. If you need to edit individual collection items, you should select them from the collection view.

Collection Items

This form only edits the collection properties, not the individual items within the collection. To modify collection items:

  1. Save any changes to the collection first
  2. Return to the collection view
  3. Select a specific item from the table to edit its properties
