Merging Dictionary Entries

This page allows you to combine two similar or duplicate dictionary entries into a single entry.

When to Merge Entries

  • When multiple entries refer to the same real-world entity
  • When duplicate entries were created by mistake
  • When entries with slight variations should be consolidated

Merging Process

  • Select a primary entry that will be kept
  • Select secondary entries to merge into the primary entry
  • Review the resulting merged entry
  • Confirm the merge operation

Important Considerations

  • Merging is permanent and cannot be easily undone
  • All tags associated with merged entries will be reassigned
  • Only entries of the same type can be merged
  • The merged entry retains the primary entry's normalized form
