Review Collections

This page allows you to systematically review collection items. The review workflow presents items one by one, letting you examine, modify, and set a status for each item before proceeding to the next.

Starting a Review Workflow

  1. Select the collection you want to review
  2. Click "Start Review Workflow" to begin
  3. Only one review workflow can be active at a time

Review Process

For each collection item, you can:

  • Review Content: Examine the item's text and annotations
  • Edit Item: Make changes to the title or content
  • Add Notes: Record comments or issues in the notes field
  • Set Status: Mark the item as:
    • Reviewed: Item is correct and complete
    • Faulty: Item has issues that need attention
    • Open: Skip for now and review later
  • Continue: Move to the next item after setting status

Workflow Completion

The workflow continues until you've reviewed all items or choose to stop. You can:

  • Pause and resume the workflow later
  • Get an overview of your progress
  • Generate a report of reviewed items when finished
