Dictionary Authority Integration

This feature allows you to link dictionary entries to external authority records, improving data quality and enabling integration with established knowledge bases.

What is Authority Integration?

Authority integration helps you:

  • Standardize entity references using established identifiers
  • Enrich entries with additional metadata from authority sources
  • Enable interoperability with other systems and datasets
  • Improve search and filtering capabilities
  • Validate and disambiguate similar entities

Integration Process

  1. Enter a search term related to your dictionary entry
  2. Browse the search results from the authority database
  3. Review detailed information about potential matches
  4. Select the most appropriate match for your entry
  5. Apply the selected authority data to your dictionary entry

When to Use

  • When creating new dictionary entries
  • When improving existing entries that lack standardization
  • When preparing data for export or sharing
  • When resolving ambiguous or duplicate entities

Best Practices

  • Use specific search terms to find better matches
  • Review all metadata before applying to your entry
  • Be consistent in how you select authority records
  • Document cases where no suitable match exists
  • Consider the scope and focus of your project when selecting matches
