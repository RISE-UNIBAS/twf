Context Help: GND Batch Enrichment

The GND Batch Enrichment page allows you to automatically enrich multiple dictionary entries with data from the Integrated Authority File (GND - Gemeinsame Normdatei) maintained by the German National Library.

What is GND?

The Gemeinsame Normdatei (GND) is a comprehensive authority file used primarily for:

  • Standardizing names of persons, organizations, events, and places
  • Providing unique identifiers (GND numbers) for entities
  • Storing biographical, geographical, and other metadata about entities
  • Connecting library and archival resources with standardized references

Batch Enrichment Process

  1. Dictionary Selection: Choose the dictionary to enrich (typically a Person, Organization, or Place dictionary)
  2. Entry Selection: Select the entries to be enriched with GND data:
    • Filter by entry status or characteristics
    • Choose all entries or a specific subset
  3. Enrichment Configuration: Set parameters for the batch process:
    • Search precision settings
    • Confidence threshold for automatic matching
    • Fields to import from GND
  4. Start Batch Process: Begin the automated enrichment

Enrichment Results

After the batch process completes:

  • Successfully Enriched: Entries that were confidently matched and updated with GND data
  • Uncertain Matches: Entries with potential matches that require manual review
  • No Matches: Entries for which no suitable GND records were found

GND Data Fields

Depending on the entity type, GND can provide:

  • For Persons: Birth/death dates, biographical information, alternative names, occupations
  • For Organizations: Founding date, location, alternative names, hierarchical relationships
  • For Places: Geographical coordinates, hierarchical relationships, historical names
  • For All Types: GND ID, related entities, external identifiers (VIAF, ISNI, etc.)

Best Practices

  • Ensure dictionary entries have clear, specific names to improve match quality
  • Start with smaller batches to test match quality before processing large sets
  • Review uncertain matches manually after batch processing
  • Consider using supervised workflows for more ambiguous entries
  • GND works best for German-language content and German-speaking regions
