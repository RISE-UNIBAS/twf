Context Help

This page allows you to view and edit your user profile. You can change:

  • Your name
  • Email address
  • Password
  • ORCID iD
  • Affiliation

The user information is used to:

  • Identify you in the system
  • Provide you with the correct permissions
  • Attach user details to exported data for curation tracking

Your password can be changed in the "Change Password" section:

  • Enter your current password
  • Enter your new password twice to confirm it
  • If forgotten, contact your system administrator

Permissions are granted on a project basis:

  • If you have project access, you will see your permissions
  • If you need access but haven't been granted it, contact your system administrator
