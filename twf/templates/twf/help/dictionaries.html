Context Help: Dictionaries

The Dictionaries page displays all dictionaries currently available in your project, allowing you to browse, search, and manage them. Here you can view details about each dictionary and access its entries.

Dictionary List

The main table on this page shows:

  • Dictionary Name: The title of each dictionary
  • Type: The category (person, place, organization, date, etc.)
  • Entries: The number of terms contained in the dictionary
  • Created/Modified: When the dictionary was created and last updated
  • Actions: Buttons for viewing, editing, or removing dictionaries

Dictionary Actions

For each dictionary, you can:

  • View: Open the dictionary to browse its entries
  • Edit: Modify the dictionary's properties (name, description, type)
  • Remove: Detach the dictionary from your project (does not delete the dictionary)

Filtering and Searching

You can filter the dictionary list by:

  • Dictionary Type: Show only person, place, or other types of dictionaries
  • Name/Description: Search for dictionaries containing specific text
  • Status: View dictionaries by their completion or usage status

Adding New Dictionaries

From this page, you can access two methods to add dictionaries to your project:

  • Add Existing Dictionary: Incorporate dictionaries that already exist in the system
  • Create New Dictionary: Start a new dictionary from scratch

Dictionary Status

Dictionaries may have different statuses:

  • Active: Currently in use for tag normalization
  • In Progress: Still being developed or enriched
  • Reference Only: Used only for lookup, not automatic normalization

Using Dictionaries

Dictionaries in your project are used to:

  • Normalize variations of the same entity in your tags
  • Provide structured information about tagged entities
  • Link your data to external knowledge bases and authority records
  • Improve search, filtering, and analysis capabilities
