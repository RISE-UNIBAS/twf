Dictionaries

This page displays all dictionaries available in your project. Dictionaries help normalize variations of the same entity and link your data to external knowledge bases.

Dictionary List

The main table shows:

  • Dictionary Name: The title of each dictionary
  • Type: The category (person, place, organization, date, etc.)
  • Entries: The number of terms in the dictionary
  • Created/Modified: When the dictionary was created and last updated

Available Actions

  • View: Open the dictionary to browse its entries
  • Edit: Modify the dictionary's properties
  • Remove: Detach the dictionary from your project
  • Add Dictionary: Incorporate existing dictionaries
  • Create Dictionary: Start a new dictionary
