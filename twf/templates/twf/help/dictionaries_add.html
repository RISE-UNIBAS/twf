Context Help: Add Dictionaries

The Add Dictionaries page allows you to incorporate existing dictionaries into your current project. This feature lets you leverage dictionaries that have already been created in the system, saving time and promoting consistency across projects.

Dictionary Selection

On this page, you will see:

  • A list of available dictionaries that are not yet part of your project
  • Details about each dictionary (name, type, entries count, creator)
  • Options to filter dictionaries by type, name, or other criteria

Adding Dictionaries to Your Project

To add a dictionary:

  1. Browse the list of available dictionaries
  2. Click the "Add" button next to dictionaries you want to include
  3. Review the confirmation message
  4. The dictionary will now appear in your project's dictionary list

Dictionary Sharing

Important points about dictionary sharing:

  • Dictionaries in TWF can be shared across multiple projects
  • When you add a dictionary, you're creating a reference to the original dictionary, not a copy
  • Changes made to shared dictionaries affect all projects that use them
  • You must have appropriate permissions to add and use shared dictionaries

Dictionary Types

When selecting dictionaries, consider the types you need:

  • Person Dictionaries: For normalizing names of individuals
  • Place Dictionaries: For geographic locations and place names
  • Organization Dictionaries: For institutions, companies, and groups
  • Date Dictionaries: For standardizing date formats and historical dates
  • General Dictionaries: For other types of terms

Best Practices

Consider these tips when adding dictionaries:

  • Start by adding established, well-maintained dictionaries
  • Check the number of entries to gauge dictionary completeness
  • Review the dictionary's creator and last modification date
  • Add dictionaries that match your project's focus and scope
  • Consider using dictionaries with external authority links
