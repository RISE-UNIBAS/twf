Add Dictionaries

This page allows you to add existing dictionaries to your current project. Dictionaries can be shared across multiple projects to maintain consistency and save time.

Using This Page

  • Browse the list of available dictionaries not yet in your project
  • View details like name, type, entries count, and creator
  • Click the "Add" button next to dictionaries you want to include

Dictionary Sharing

  • When you add a dictionary, you're creating a reference to the original
  • Changes to shared dictionaries affect all projects using them
  • Dictionary types include Person, Place, Organization, Date, and General

Tips

  • Add dictionaries that match your project's focus
  • Check the entry count to gauge dictionary completeness
  • Consider dictionaries with external authority links
