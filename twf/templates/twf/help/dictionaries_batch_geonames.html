Context Help: GeoNames Batch Enrichment

The GeoNames Batch Enrichment page allows you to automatically enrich multiple dictionary entries with data from GeoNames, a geographical database containing information about places and locations worldwide.

What is GeoNames?

GeoNames is a global geographical database that:

  • Contains over 25 million geographical names and features
  • Provides unique identifiers for each location
  • Includes latitude, longitude, elevation, population, and other data
  • Covers administrative divisions, natural features, and populated places
  • Is available in multiple languages
  • Offers a hierarchical structure of locations (country > region > city)

Batch Enrichment Process

  1. Dictionary Selection: Choose the dictionary to enrich (typically a Place dictionary)
  2. Entry Selection: Select the entries to be enriched with GeoNames data:
    • Filter by entry status or characteristics
    • Choose all entries or a specific subset
  3. Enrichment Configuration: Set parameters for the batch process:
    • Search precision settings
    • Geographical focus areas (countries or regions)
    • Feature types to include (cities, natural features, etc.)
    • Confidence threshold for automatic matching
  4. Start Batch Process: Begin the automated enrichment

Enrichment Results

After the batch process completes:

  • Successfully Enriched: Entries that were confidently matched and updated with GeoNames data
  • Uncertain Matches: Entries with potential matches that require manual review
  • No Matches: Entries for which no suitable GeoNames records were found

GeoNames Data Fields

GeoNames can provide the following data for place entries:

  • Coordinates: Precise latitude and longitude
  • Administrative Hierarchy: Country, region, sub-region
  • Alternative Names: Names in different languages and historical names
  • Feature Class: Type of geographical feature (city, mountain, lake, etc.)
  • Population: For populated places
  • Elevation: For relevant features
  • Time Zone: Local time zone information

Best Practices

  • Include country or regional context in place names when possible
  • Specify geographical focus areas to reduce ambiguity
  • Start with smaller batches to test match quality
  • Review uncertain matches manually after batch processing
  • Consider historical place names which may require special handling
  • Use supervised workflows for places with common names

Requirements

To use GeoNames enrichment, your project must have valid GeoNames API credentials configured in the project settings.
