Creating a New Project

As an administrator, you can create a new project by clicking on the "Create Project" button in the navigation bar.

Project Details

Each project requires the following details:

  • Title: The project's title must be unique in the database.
  • Description: A brief description of the project that can be included in exports.
  • Transkribus Collection ID:
    • The ID of the collection in Transkribus, used to link the project to the collection.
    • If no collection exists in Transkribus, enter 0.
    • TWF can handle non-Transkribus data, but it is strongly recommended to start your project in Transkribus.

User Roles & Permissions

Each project has assigned roles that define user permissions:

  • Project Owner: The user assigned as the project owner. Can edit the project and manage users. Does not have to be an admin account. Permissions are assigned per project.
  • Project Members: Users who are granted access to the project. They can view the project and work within workflows. Users can be added or removed later.
