Context Help

As an administrator, you can create a new project by clicking on the "Create Project" button in the navigation bar.

  • Title: The project's title must be unique in the database.
  • Description: A brief description of the project. Can be used in exports.
  • Transkribus Collection ID: The ID of the collection in Transkribus. This is used to link the project to the collection in Transkribus. If you don't have a collection in Transkribus, you can enter 0 to this field. TWF can deal with Non-Transkribus data, but it is strongly advised to start your project in Transkribus.
  • Project owner: The user who will be the owner of the project. The owner can edit the project and assign other users to the project. This does not have to be an admin account. Permissions are given on a per-project basis.
  • Project members: You can add other users to the project. They will be able to view the project and get basic credentials to work workflows. You can add or emove users later.
