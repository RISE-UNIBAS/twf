Context Help: Export Dictionaries

The Export Dictionaries page allows you to export dictionaries from your project. Dictionaries are collections of terms, definitions, and normalized values that help standardize tags and terms across your project.

Export Process

  1. Select Dictionaries: Choose which dictionaries to include in the export.
  2. Choose Format: Select the desired export format (JSON, CSV, Excel, etc.).
  3. Configure Options: Set specific options for the selected format, such as:
    • Including dictionary metadata
    • Including variations and normalized forms
    • Including external references
    • Field mapping options
  4. Start Export: Click the export button to begin the export process.

Dictionary Content

When exporting dictionaries, you can choose to include:

  • Dictionary Metadata: Information about the dictionary itself, such as type, description, and creation date.
  • Dictionary Entries: All entries contained in the dictionary.
  • Variations: Different forms or spellings of each entry.
  • Normalized Values: The standardized form for each entry.
  • External References: Links to external authorities like GND, Wikidata, or Geonames.
  • Usage Statistics: Information about how frequently entries appear in your documents.

Use Cases for Dictionary Exports

  • Data Sharing: Share standardized terminology with colleagues or other projects.
  • Backup: Create backups of your dictionaries.
  • External Enrichment: Export to enrich the data elsewhere before re-importing.
  • Publication: Include standardized dictionaries as part of your research data publication.
