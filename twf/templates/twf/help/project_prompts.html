Context Help: Saved Prompts

The Saved Prompts page allows you to manage a library of reusable AI prompts for your project. This feature helps maintain consistency in AI interactions and saves time when performing repeated operations.

What Are Saved Prompts?

Saved prompts are pre-defined instructions or questions that you can reuse when interacting with AI services (OpenAI, Claude, Gemini) within your project. They can include:

  • Questions for data analysis
  • Instructions for content generation
  • Templates for standardized AI tasks
  • Custom workflows for specific project needs

Managing Prompts

On this page, you can:

  • View: Browse all saved prompts in your project.
  • Create: Add new prompts with titles and descriptions.
  • Edit: Modify existing prompts to improve their effectiveness.
  • Delete: Remove prompts that are no longer needed.
  • Organize: Group prompts by categories or tags.

Creating Effective Prompts

When creating prompts, consider these best practices:

  • Be clear and specific about what you're asking the AI to do.
  • Include placeholders for variable information (e.g., [DOCUMENT_NAME]).
  • Use descriptive titles that indicate the prompt's purpose.
  • Add notes about when and how to use each prompt.
  • Test prompts with different data sources to ensure versatility.

Prompt Variables

You can use special variables in your prompts that will be automatically replaced when used:

  • {document}: The current document's content
  • {page}: The current page's content
  • {collection}: The current collection's content
  • {tags}: Tag information
  • {project}: Project information

Collaborative Prompt Management

Saved prompts are available to all project members (with appropriate permissions), enabling:

  • Consistent AI usage across the team
  • Knowledge sharing of effective prompt strategies
  • Standardized workflows for recurring tasks
