Editing Dictionary Entries

This page allows you to edit dictionary entries, which represent normalized entities such as people, places, or organizations.

Key Fields

  • Label: Primary identifier, typically the canonical form
  • Type: Category (person, place, organization, etc.)
  • Normalized Form: Standardized version linked to external authorities
  • Definition: Optional description of the entry
  • Metadata: Additional structured information

Working with Normalized Forms

  • Search external authorities (GND, Wikidata, GeoNames)
  • Select from search results or create a manual form
  • View and modify normalized form details as needed

Best Practices

  • Be consistent with formatting labels and names
  • Link to external authorities when possible
  • Add sufficient context to distinguish similar entries
  • Review associated tags after making significant changes
