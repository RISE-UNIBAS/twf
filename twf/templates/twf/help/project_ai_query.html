AI Query

The AI Query feature allows you to ask questions about your project data and receive intelligent responses from AI language models. This is useful for exploring your data, generating summaries, extracting insights, or getting assistance with analysis.

How AI Querying Works

When you use this feature:

  • Your prompt and selected data are sent securely to the AI service (OpenAI, Claude, or Gemini).
  • The AI processes your query and returns a response based on the provided data.
  • Results are displayed in the interface and can be saved for future reference.
  • Important: Querying data will not change your project data in any way.

Configuration Requirements

To use this feature, you need to:

  • Have valid API credentials configured in your project settings.
  • Select an appropriate AI model from the available options.
  • Have necessary permissions within your project.

Writing Effective Prompts

For best results:

  • Be specific about what you want the AI to analyze or explain.
  • Provide context about your data when needed.
  • Break complex questions into simpler parts.
  • Use the "Save Prompt" feature to store useful prompts for reuse.

Data Selection

You can query different types of project data:

  • Documents (full text or specific pages)
  • Collections
  • Tags and annotations
  • Dictionary entries
  • Project metadata

Privacy & Security

Please be aware that:

  • Data sent to AI services is governed by their privacy policies.
  • Sensitive information should be handled with appropriate caution.
  • API keys should be kept secure and not shared.
