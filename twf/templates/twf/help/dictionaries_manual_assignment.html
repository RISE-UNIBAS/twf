Manual Dictionary Assignment

This feature allows you to manually assign tags and entities to appropriate dictionary entries. Manual assignment gives you complete control over how entities are categorized and standardized in your project.

When to Use Manual Assignment

Manual dictionary assignment is particularly valuable when:

  • Working with specialized or complex entities that AI might misinterpret
  • Handling ambiguous terms that require human judgment
  • Ensuring high precision in critical dictionary entries
  • Creating initial high-quality examples for later automated processes
  • Working with historical, obscure, or context-dependent terms
  • Applying domain expertise that automated systems lack

The Manual Assignment Process

To manually assign entities to dictionaries:

  1. Select a tag or entity that needs dictionary assignment
  2. Choose the appropriate dictionary for this entity type
  3. Search for matching existing entries or create a new entry
  4. Review the proposed assignment for accuracy
  5. Confirm the assignment to link the entity to the dictionary entry
  6. Add additional information or normalization if needed

Assignment Options

When manually assigning entities, you have several options:

  • Assign to existing entry: Link the entity to a dictionary entry that already exists
  • Create new entry: Establish a new dictionary entry for this entity
  • Create variant: Add this entity as a variant of an existing entry
  • Merge entries: Combine similar entities into a single dictionary entry
  • Assign with context: Include contextual information with the assignment

Search and Matching

The system provides several tools to help you find appropriate dictionary entries:

  • Fuzzy search: Find entries that approximately match your entity
  • Similarity suggestions: View entries that might be related
  • Recent assignments: See recently used dictionary entries
  • Filtered views: Narrow down entries by various criteria
  • Sorting options: Order entries by relevance, frequency, or alphabetically

Creating New Entries

When creating a new dictionary entry during manual assignment, you can:

  • Set the canonical form of the entity
  • Provide additional metadata and descriptive information
  • Link to external authority records or identifiers
  • Add notes about usage or special cases
  • Establish relationships with other dictionary entries
  • Configure normalization settings for this entry

Best Practices

  • Establish clear guidelines for manual assignment before starting
  • Be consistent in how you handle similar entities
  • Document special cases or exceptions
  • Check for existing entries before creating new ones
  • Add sufficient context to disambiguate similar entities
  • Review assigned entities periodically for consistency
  • Consider using batch assignment for obvious cases, saving manual work for complex situations
