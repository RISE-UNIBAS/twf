Project Notes

The Project Notes feature allows you to create, manage, and share important notes related to your project. Notes are a convenient way to document decisions, record important information, and communicate with team members.

Notes Overview

On this page, you can:

  • View all notes associated with your current project
  • Filter notes based on various criteria
  • Access detailed views of individual notes
  • Create new notes to share information with your team

Notes Table

The main section of the page displays a table of all notes in your project. For each note, you can see:

  • Title: The note's title, which serves as a quick reference
  • Created: When the note was initially created
  • Created By: The user who created the note
  • Last Modified: When the note was last updated
  • Actions: Options to view, edit, or delete the note

Filtering Notes

Use the filters at the top of the page to narrow down the list of notes. You can filter by:

  • Title: Find notes containing specific text in their titles
  • Creator: Show notes created by a specific team member
  • Date Range: Display notes created or modified within a certain timeframe

After setting your filters, the table will update to show only the matching notes, with a count of how many notes were found.

Working with Notes

Creating a Note

To create a new note:

  1. Click the "+" button in the upper right corner of the page
  2. Enter a title that clearly describes the note's content
  3. Type your note text in the provided text area
  4. Click "Save Note" to create the note

Viewing a Note

To view a note's full details:

  1. Find the note in the table
  2. Click the "View" button (eye icon) for that note
  3. The note details page will show the complete note content and metadata

Editing a Note

To modify an existing note:

  1. View the note details page
  2. Click the "Edit Note" button
  3. Make your changes to the title or content
  4. Click "Save Note" to update the note

Deleting a Note

To remove a note that is no longer needed:

  1. View the note details page
  2. Click the "Delete Note" button
  3. Confirm your decision in the confirmation dialog

Note: Deleting a note is permanent and cannot be undone.

Best Practices for Notes

  • Use Clear Titles: Make titles descriptive so team members can quickly understand the note's purpose
  • Keep Notes Organized: Create separate notes for different topics rather than combining multiple subjects
  • Include Context: Provide enough background information for others to understand the note's content
  • Update Regularly: Edit notes when information changes rather than creating duplicate notes
  • Document Decisions: Use notes to record important project decisions and their rationale
