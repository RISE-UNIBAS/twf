Context Help: Export Documents

The Export Documents page allows you to export documents from your project in various formats. You can select specific documents to export or filter them based on criteria.

Export Process

  1. Select Documents: Use the table to select which documents to include in the export.
  2. Choose Format: Select the desired export format (JSON, XML, CSV, TXT, etc.).
  3. Configure Options: Set specific options for the selected format, such as:
    • Including page content
    • Including metadata
    • Including tags
    • Formatting options
  4. Start Export: Click the export button to begin the export process.

Supported Formats

Depending on your project configuration, you may export documents in various formats:

  • JSON: Best for preserving all data structures and metadata.
  • XML: Good for interoperability with other systems.
  • TXT/CSV: Simplified formats for text content and basic metadata.
  • PDF: For human-readable document exports.
  • Custom formats: Defined in your project's export configuration.

Document Selection Tips

  • Use filters to quickly find documents based on metadata, tags, or other criteria.
  • You can sort the document table by any column to organize your selection.
  • Select all documents that match your current filter with the "Select All" checkbox.
