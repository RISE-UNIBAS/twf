Collection Batch Processing

This feature allows you to process multiple collection items at once using AI models. Batch processing is useful for analyzing, categorizing, or enriching collections of related documents.

Prerequisites

  • A valid API key configured in your project settings
  • A collection with properly configured items
  • Defined prompts that specify what the AI should analyze or extract

Batch Processing Steps

  1. Select the collection to process
  2. Select or create a prompt template
  3. Configure processing parameters
  4. Start the batch process
  5. Monitor progress and review results
