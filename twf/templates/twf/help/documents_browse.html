Context Help: Browse Documents

The Browse Documents page allows you to view, search, and filter all documents in your project. This is your primary interface for exploring document content and accessing individual documents for detailed viewing.

Document Browsing Features

On this page, you can:

  • View Document List: See all documents in your project with key information
  • Search Documents: Find documents by title, content, or metadata
  • Filter Documents: Narrow down the list using various criteria
  • Sort Documents: Order documents by different properties
  • Access Document Details: Open individual documents for detailed viewing

Document List Information

The document list table displays:

  • Document Title: The name or title of each document
  • Document ID: Unique identifier from Transkribus or other source
  • Pages: Number of pages in the document
  • Status: Current workflow status (new, in progress, reviewed, etc.)
  • Tags: Count of tags extracted from the document
  • Created/Modified: When the document was added and last updated
  • Actions: Buttons for viewing and managing each document

Search and Filter Options

You can narrow down the document list using:

  • Text Search: Find documents containing specific text
  • Metadata Filters: Filter by document properties
  • Status Filters: Show only documents in certain workflow stages
  • Date Range: View documents created or modified within specific periods
  • Tag Filters: Find documents containing specific tag types

Document Actions

For each document, you can:

  • View: Open the document to see its content and details
  • Edit Metadata: Modify document information
  • Add to Collection: Include the document in a collection
  • Export: Generate exports of individual documents
  • Change Status: Update the document's workflow status

Batch Operations

You can also perform actions on multiple documents at once:

  • Select multiple documents using checkboxes
  • Apply batch operations from the action menu
  • Process selected documents with AI tools

Navigation Tips

  • Use the search box for quick document location
  • Save frequently used filter combinations for reuse
  • Sort by different columns to organize your view
  • Adjust the number of documents shown per page
  • Use pagination controls to navigate through large document sets
