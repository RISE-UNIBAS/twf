Context Help: Documents Overview

The Documents Overview page serves as the central hub for accessing and managing all documents within your project. Documents are the primary content containers in TWF, typically representing individual files from Transkribus exports.

What Are Documents?

In TWF, documents:

  • Contain transcribed text from original source materials
  • Are organized into pages, which may include images and text
  • Store metadata about the source material
  • Can be enriched with tags, annotations, and normalized data
  • Serve as the foundation for collections and further analysis

Document Management Features

From this page, you can access:

  • Browse Documents: View and search through all documents in your project
  • Document Creation: Create new documents manually if needed
  • Document Review: Access the review workflow for documents
  • Name Documents: Assign standardized names to documents
  • AI Workflows: Process documents with AI tools (OpenAI, Claude, Gemini)

Document Statistics

The overview page shows key information about your document collection:

  • Total number of documents in the project
  • Documents by status (reviewed, in progress, etc.)
  • Recently accessed or modified documents
  • Page count and other aggregated statistics

Document Workflow Options

TWF provides several ways to work with your documents:

  • Manual Workflows: Review, edit, and manage documents individually
  • Batch Processing: Apply operations to multiple documents at once
  • AI-Assisted Analysis: Use AI models to analyze document content
  • Collection Creation: Organize documents into thematic collections

Document Management Best Practices

  • Establish consistent naming conventions for documents
  • Extract and normalize tags early in your workflow
  • Use the document review process to ensure quality
  • Add relevant metadata to improve searchability
  • Consider document relationships when creating collections
