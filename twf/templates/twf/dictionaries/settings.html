Configuration Guide

Enrichment Types: Select which enrichment types are available for each dictionary type (click to select multiple):

  • Date: Normalize dates to EDTF format
  • Bible Verse: Normalize biblical references
  • Authority ID: Generic authority control ID enrichment
  • GND: German National Library authority control for persons, places, and organizations
  • Wikidata: Linked open data from Wikidata (requires entity type configuration)
  • GeoNames: Geographic database for location enrichment

Wikidata Entity Type: When using Wikidata enrichment, specify the entity type (person, city, etc.) to improve search results. This field is only enabled when Wikidata is selected.

Workflow Title: Custom title displayed when starting manual enrichment workflows for this dictionary type.
