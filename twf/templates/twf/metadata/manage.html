About Metadata Blocks

What are metadata blocks?

Metadata blocks are top-level keys in the document or page metadata JSON. For example, if you imported JSON metadata with a storage key of json_import, that becomes a metadata block.

Common metadata blocks:

  • json_import - Default key for JSON metadata imports
  • google_sheets - Metadata loaded from Google Sheets
  • transkribus_metadata - Metadata enriched from Transkribus API

How to delete a block:

  1. Select whether you want to delete from documents or pages
  2. Choose the metadata block to delete from the dropdown
  3. Check the confirmation box
  4. Click "Delete Metadata Block"

What happens when you delete?

The selected metadata key and all its nested data will be removed from every document or page in this project. Other metadata blocks will remain untouched.
