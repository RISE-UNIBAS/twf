Configuration Guide

Review Workflows: Select which metadata fields to include in document and page review workflows.

Field Configuration:

  • Include: Check to include this field in the review form
  • Metadata Key: The path to the metadata field (e.g., "json_import.author")
  • Count: Number of documents/pages with this field
  • Field Label: Display name shown in the review form
  • Field Type: Input type (text, textarea, number, date, select)

Usage: After configuring fields here, they will appear in the Document/Page Review workflows where you can edit metadata values in a batch workflow.
