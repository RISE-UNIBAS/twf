Configuration Guide

Page Metadata Review: Configure which metadata fields appear in the page metadata review workflow.

Document Metadata Review: Configure which metadata fields appear in the document metadata review workflow.

JSON Format: Both configurations use JSON format with the following structure:

{
  "field.path": {
    "label": "Display Name",
    "type": "text"
  }
}

Field Types:

  • text - Single line text input
  • textarea - Multi-line text input
  • number - Numeric input
  • date - Date picker
  • select - Dropdown (requires options)

Example Configuration:

{
  "json_import.author": {
    "label": "Author",
    "type": "text"
  },
  "json_import.title": {
    "label": "Title",
    "type": "text"
  },
  "json_import.year": {
    "label": "Year",
    "type": "number"
  }
}
