A Bit Dangerous

These actions are not irreversible but still modify your data significantly.

Unpark all tags: All parked tags in the project are un-parked.

Remove all saved prompts: Removes all saved prompts from the project

Remove all tasks: All tasks are deleted. This will not alter your data but remove the history of performed tasks.

Remove all dictionaries: Dictionaries are not deleted but removed from the project.

Dangerous

These actions are irreversible. Proceed with caution.

Delete all documents: This deletes all document objects in your project and subsequently all page and tag objects. Collections can no longer be edited.

Delete all tags: This deletes all tags. They can be re-extracted from Transkribus document data.

Delete all collections: This deletes all collections in the project.
