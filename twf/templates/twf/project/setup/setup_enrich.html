This page enables you to enrich your documents with additional metadata from the Transkribus API that is not available in the PageXML export.

What does this do?

This task fetches additional metadata for your documents from the Transkribus API:

  • Document Labels: Tags and labels applied at the document level
  • Page Labels: Labels applied to individual pages (e.g., "Exclude")
  • Exclusion Status: Automatically marks pages with "Exclude" label as ignored

Note: This data is stored in the metadata fields and can be used for filtering and organization.
