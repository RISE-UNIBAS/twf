Setup Project

Import Documents from Transkribus or JSON data.

Transkribus Import

To import your documents into the TWF, you'll need to create a Transkribus export. This does not work manually, so you'll have to enter your Transkribus credentials and your collection id in the project settings. in the Project Settings and start an export from Transkribus here. Depending on your collection, this may take a while. When the export is finished, you can download it to the server and extract the Data

JSON Import

You can import a list of json objects. Each object is going to form a document. Each document object can contain a list of page objects. The corresponding JSON data is copied to the metadata fields of the documents and pages.

Please note that JSON import does not support IIIF functionality and doesn't link to any source collection.

Google Sheets Import

You can import document metadata you have stored in a Google sheet document. You'll need to enter the Google sheet id and other necessary settings in the Project Settings.

Your Documents have an id (which is not the TWF-database id but the id of the transkribus document or if you imported json data the one you provided when importing). This id needs to be present in the Google sheets document in order for this to work.
