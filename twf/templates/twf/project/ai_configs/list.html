What are AI Configurations?

AI Configurations bundle everything needed to make an AI call:

  • Provider & Model: Choose from OpenAI, Claude, Gemini, and more
  • Credentials: Store API keys securely with each configuration
  • Prompt Template: Define reusable prompts with placeholders
  • System Role: Set the AI's behavior and personality
  • Execution Settings: Control temperature, tokens, and other parameters

Once created, configurations can be selected in workflows for document analysis, tag enrichment, collection processing, and more.
