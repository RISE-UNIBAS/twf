Configuration Guide

Dictionary Type: Connect tag types to dictionary types. For example, both "sender_person" and "recipient_person" can map to "person" dictionary.

Workflow:

  • Group: Tag variations are grouped and assigned to dictionary entries (default for most tags)
  • Enrich: Tags receive direct enrichment without grouping (e.g., dates, Bible verses)
  • Ignore: Tags are excluded from workflows and processing

Enrichment Configuration: For "Enrich" workflow types, specify the workflow title and form type (e.g., "verse", "date").
