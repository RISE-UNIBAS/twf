Project AI Views
================

The Project AI views provide interfaces for querying various AI models with content
from the project's documents. These views support text-only and multimodal interactions
depending on the AI provider's capabilities.

Supported AI Providers:
- OpenAI (GPT-4, GPT-4 Vision)
- Google Gemini
- Anthropic Claude
- Mistral
